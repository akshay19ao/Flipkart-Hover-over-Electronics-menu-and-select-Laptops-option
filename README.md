# Flipkart-Hover-over-Electronics-menu-and-select-Laptops-option
• Open Flipkart homepage.
• Close the login popup if displayed.
• Hover over the “Electronics” menu.
• Click on “Laptops” option.
• Verify the page title contains “Laptops”.
Expected Outcome: Laptops product page is opened, and the title contains “Laptops”.
